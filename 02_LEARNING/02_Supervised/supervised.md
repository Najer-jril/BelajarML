# Supervised Learning

Supervised learning adalah metode machine learning dimana model belajar dari data berlabel untuk memprediksi output berdasarkan input yang diberikan.

## Topik yang Akan Dipelajari

### Regression
- Linear Regression
- Polynomial Regression
- Ridge dan Lasso Regression
- Evaluation metrics: MSE, RMSE, MAE, R-squared

### Classification
- Logistic Regression
- Decision Trees
- Random Forest
- Support Vector Machines (SVM)
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Evaluation metrics: Accuracy, Precision, Recall, F1-Score, Confusion Matrix, ROC-AUC

### Advanced Topics
- Gradient Boosting (XGBoost, LightGBM, CatBoost)
- Neural Networks untuk supervised learning
- Cross-validation techniques
- Hyperparameter tuning
- Feature engineering
- Handling imbalanced data

## Dataset yang Digunakan

Repository ini memiliki dataset properti dari berbagai kota di Belanda:
- Amsterdam, Delft, Den Haag, Eindhoven, Groningen, Rotterdam, Utrecht

Dataset ini dapat digunakan untuk:
- Prediksi harga sewa (regression)
- Klasifikasi tipe properti (classification)
- Feature analysis untuk faktor yang mempengaruhi harga

## Struktur Pembelajaran

1. Eksplorasi dan preprocessing data
2. Feature selection dan engineering
3. Model training dan validation
4. Model evaluation dan comparison
5. Model optimization
6. Deployment considerations
